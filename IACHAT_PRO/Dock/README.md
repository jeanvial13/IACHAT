# IACHAT_PRO — Terminal UI + ZIPs + Créditos

Despliegue:
1) Copia este repo a GitHub.
2) En Portainer -> Stacks -> Add Stack -> Repository
   - Repository URL: https://github.com/<tu_usuario>/IACHAT_PRO.git
   - Compose path: Dock/docker-compose.yml
3) En 'Environment variables' agrega OPENAI_API_KEY y opcional OPENAI_MODEL.
4) Deploy. Abre http://<IP-NAS>:8080
